# 🏦 AI Forex Trading Assistant — منصة التداول الذكية

منصة تداول مؤسسية كاملة (وليست مجرد شات بوت) مبنية على Gemini AI، بمبدأ أساسي واحد:
**حماية رأس المال أولاً (Capital Protection First) — الربح ثانياً.**

المساعد يتكلم **بالعامية المصرية** مع المصطلحات الإنجليزية بين أقواس، ويرفض أي صفقة ضعيفة (Strict Mode دائماً مفعّل — ثقة أقل من 70 = NO TRADE).

---

## ✅ المميزات المكتملة

### 1. التحليل الذكي (AI Analysis) — `index.html` → تبويب "التحليل الذكي"
- رفع صور متعددة (حتى 8) بالسحب والإفلات، اللصق (Ctrl+V)، أو الاختيار — تُضغط تلقائياً قبل الإرسال
- دعم صور TradingView وBookmap ومختلط — تُحلل كتحليل واحد مدمج (Multi-Image Merge)
- اختيار السوق، الفريم، ونوع الطلب (صفقة جديدة / تحديث / إدارة صفقة / اتجاه / تعليم / أخبار)
- **محادثة حية (Live Chat)** بذاكرة جلسة كاملة — المتابعة والأسئلة بدون إعادة التحليل
- **System Prompt مؤسسي كامل** (`js/prompt.js`): SMC، ICT، Liquidity، Bookmap، Volume، Wyckoff، محرك القرار بالتجميع (Confluence Scoring 0-100)، قواعد SL/TP/Breakeven/Partial Close/Invalidation، منع الهلوسة (لا اختراع BOS/CHOCH/OB/FVG/أخبار)
- عرض القرار النهائي كشريحة ملونة + شريط الثقة (Confidence Bar)
- استخراج JSON من نهاية كل تحليل وحفظه تلقائياً
- **نظام النسخ (Versioning)**: تحديث تحليل سابق يقارن ويشرح إيه اللي اتغير ويحفظ نسخة جديدة

### 2. Gemini API (`js/gemini.js`)
- استدعاء مباشر من المتصفح — المفتاح يُحفظ في `localStorage` فقط ولا يُرسل لأي طرف غير Google
- موديل افتراضي + موديل احتياطي (Fallback) تلقائي عند 429/404/5xx
- اختبار اتصال من الإعدادات + رسائل خطأ عربية ودّية (Quota / Invalid Key / Network / Timeout)

### 3. لوحة التحكم (Dashboard)
- إحصائيات: الرصيد، نسبة الفوز (Win Rate)، متوسط RR، المخاطرة المفتوحة
- الصفقات المفتوحة، آخر التحليلات، قائمة المتابعة، رسم بياني للأداء التراكمي (Chart.js)
- **لوحة المخاطر (Risk Dashboard)**: P/L اليوم، حد الخسارة اليومي، الحالة (آمن/حذر/أوقف التداول)

### 4. إدارة الصفقات (Trade Manager)
- إضافة/تعديل/حذف صفقات (Market/Limit/Stop، SL، TP1-3، لوت، حالة، نتيجة، P/L)
- حساب RR تلقائي، بحث وفلترة، الصفقات المفتوحة تُرسل تلقائياً كسياق لكل تحليل AI

### 5. حاسبة المخاطرة (Risk Calculator) — `js/risk.js`
- حساب اللوت لـ XAUUSD، XAGUSD، أزواج الفوركس الرئيسية، BTC/ETH، US30/NAS100/SPX500، النفط + وضع مخصص
- المخاطرة بالدولار، Pips، أرباح TP1-3 مع RR لكل هدف
- تحقق صارم: ❌ رفض الصفقة لو RR أقل من حدك الأدنى، تحذير فوق 2-3% مخاطرة

### 6. يومية التداول (Journal)
- سبب الدخول/الخروج، الأخطاء، الدروس، **الحالة النفسية** (خوف/طمع/انتقام/انضباط...)، النتيجة، بحث كامل

### 7. قائمة المتابعة (Watchlist)
- رموز غير محدودة — تتحدث تلقائياً بالانحياز (Bias) والثقة من آخر تحليل AI

### 8. سجل التحليلات (History)
- كل تحليل محفوظ بنسخه — عرض كامل + زر "استكمال/تحديث" يكمل الجلسة من حيث توقفت

### 9. TradingView مباشر
- شارت مدمج (Advanced Chart Widget) مع تغيير الرمز والفريم

### 10. الإعدادات + التصدير
- API Key (إظهار/إخفاء/حذف/اختبار)، الموديلات، ملف المخاطرة (رصيد، %، حد يومي، Min RR)، Strict/Coach Mode
- تصدير JSON كامل (بدون المفتاح) + تصدير الصفقات CSV

## 🔗 نقاط الدخول (Entry URIs)
| المسار | الوصف |
|---|---|
| `index.html` | التطبيق كاملاً (SPA بتبويبات) |
| `tables/trades` | REST API — الصفقات |
| `tables/analyses` | REST API — التحليلات (بالنسخ) |
| `tables/journal` | REST API — اليومية |
| `tables/watchlist` | REST API — المتابعة |

## 🗄️ نماذج البيانات
- **trades**: market, direction, order_type, entry, stop_loss, tp1-3, lot_size, risk_reward, status, result, pnl, notes
- **analyses**: market, timeframe, decision, confidence, bias, entry/sl/tp1-3, risk_reward, summary (النص الكامل), version, parent_id, model_used
- **journal**: market, entry_reason, exit_reason, mistakes, lessons, emotion, result
- **watchlist**: symbol, bias, confidence, trend, last_analysis
- **الإعدادات**: `localStorage` (المفتاح لا يغادر المتصفح إلا لـ Google)

## 🛠️ التقنيات
HTML5 + CSS3 (Dark Glass UI, RTL) · Vanilla JS · Gemini API (generateContent + inline images) · Chart.js · marked.js · TradingView Widget · Font Awesome · خط Cairo · RESTful Table API

## ⚠️ حدود معروفة (بحكم كونه موقع ستاتيك)
- **لا يوجد وصول إنترنت حي للـ AI**: الأخبار والأسعار الحية تُدخل يدوياً في الملاحظات — البرومبت مصمم ليصرّح بذلك بدلاً من الاختراع
- قيم النقطة لبعض الأزواج (USDJPY/USDCAD) تقريبية — راجع وسيطك
- ذاكرة المحادثة للجلسة الحالية؛ التحليلات المحفوظة هي الذاكرة الدائمة

## 🚀 خطوات مقترحة قادمة
1. ربط تحليل بصور محفوظة (Image History) في السجل
2. استيراد JSON/CSV
3. تنبيهات الأخبار عبر إدخال يدوي للتقويم الاقتصادي
4. تحليل الأنماط النفسية من اليومية عبر AI
5. إحصائيات أعمق (أفضل جلسة/فريم/سوق)

## 📖 طريقة الاستخدام
1. الإعدادات → أدخل مفتاح Gemini API (مجاني من [Google AI Studio](https://aistudio.google.com/apikey)) → اختبار الاتصال → حفظ
2. أدخل رصيدك ونسبة مخاطرتك
3. التحليل الذكي → ارفع صور الشارت (الفريم الأعلى أولاً) → اكتب السعر الحالي وأي أخبار → حلّل الآن
4. راجع القرار والثقة — لو أقل من 70: **NO TRADE** 🛡️

**للنشر:** استخدم تبويب **Publish** لنشر الموقع بضغطة واحدة.
